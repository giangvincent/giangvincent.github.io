FAQs


1. How does Giang protect my privacy?
We, Giang , understand the importance of your privacy. This Privacy Policy is designed to inform you on how we use the information we collect and how we protect your data. We DO NOT STORE any personal information, and we DO NOT SELL your data to any third-party company.

2. I don’t want to play Dual n-back anymore. How do I remove this game?
From your News Feed, click the arrow found at the top right. Click Settings.

Click Instant Games found in the left menu. Click the box next to the game you’d like to remove.

Click Remove.

A pop-up message will appear. Check the box to delete your game history. Click Remove.

2. Information collected while the user plays Dual n-back
2.1 To use the Service, the user may log in through their Facebook Messenger account. Through this log in process, Dual n-back collects the following information:
2.1.1 The user’s game-scoped ID
2.1.2 The user’s Facebook Messenger user ID
2.1.3 The user’s language setting
2.1.4 The timestamp of when the user last accessed the application

2.2 Upon registration, user’s game-scope ID, Facebook Messenger user ID, and user’s language setting are collected enabling Dual n-back to send push notifications sent through Facebook Messenger Game Bots.

2.3 Personal information that identifies the user such as name and profile picture are NEVER stored or cached anywhere, but only used to create customized user experience such as personalized share images.

2.4 In order to manage the information Dual n-back receives through Facebook, the user must follow the instructions provided by Facebook for modifying said user’s information and privacy settings. The privacy management tools for applications on Facebook can be found at https://www.facebook.com/settings?tab=instant_games

4. Push Notifications
4.1 Users are provided information about the Service and pertinent content through messages sent through Facebook Game Messenger Bots (hereinafter referred to as “push notifications”).

4.2 Push notifications are sent through Facebook messenger, and are set with consent from the user. The user’s browser and/or app and/or device must also support such push notifications. In order to facilitate the delivery of such push notifications, Dual n-back must store the user’s game-scoped ID, Facebook messenger user ID, user’s language setting, and timestamp of last access which are all provided when the user plays the game for the very first time.

4.3 The user may choose to opt out of the push notification service at any time by disabling the function through their Facebook Messenger settings.

5. User’s Right to Revoke Data Access
5.1 Users can always choose to remove the game from their list of active Instant Games which can be found at https://www.facebook.com/settings?tab=instant_games

5.2 Upon revoking data access, Dual n-back will not be able to make additional requests for user data or send messages to the user.

5.3 Users may also opt to delete their game history on Facebook for Dual n-back by checking the box provided in the deletion process.
